== Description ==

This Digg plugin allows your users to share your posts through Digg - With almost instant setup!

The setup for this plugin is very easy and quick - Simply enable the plugin in your Wordpress admin panel and then the plugin will be active! No more work required.

== Installation ==

1. Upload `/jr-digg.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR Digg Settings in your Administration Panel
4. Done!
